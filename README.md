## Description

School Attendance API Service built with [Nest](https://github.com/nestjs/nest).

## Requirements
Node 20.11.1

## Installation

```bash
$ npm install
```

## Migrate DB
```bash
$ prisma migrate dev
```

## Reset migration DB
```bash
$ prisma migrate reset
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
